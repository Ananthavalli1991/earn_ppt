Sample: KPI Computation (Python)

from dataclasses import dataclass

@dataclass
class KPI:
    revenue: float
    cogs: float
    shares_out: float

    def gross_margin_pct(self) -> float:
        return (self.revenue - self.cogs) / self.revenue * 100

    def eps(self, net_income: float) -> float:
        return net_income / self.shares_out

k = KPI(revenue=4_200_000_000, cogs=1_806_000_000, shares_out=1_950_000_000)
print(round(k.gross_margin_pct(), 2), "%")

Financial Math

Net Present Value (NPV):

\[ \operatorname{NPV} = \sum_{t=1}^{T} \frac{C_t}{(1+r)^t} - C_0 \]

Compound Annual Growth Rate (CAGR):

\[ \operatorname{CAGR} = \left(\frac{\text{Ending}}{\text{Beginning}}\right)^{\tfrac{1}{n}} - 1 \]
